# Website giới thiệu, bán máy PC trực tuyến

## 1. Giới thiệu dự án

Đây là dự án xây dựng website giới thiệu và bán máy PC trực tuyến cho học phần Hệ thống và Công nghệ Web.

Website được xây dựng bằng HTML5, CSS3, JavaScript, Bootstrap 5 local và jQuery local. Dự án không sử dụng backend, không sử dụng database và không dùng CDN online.

## 2. Thành viên và phân công

| Thành viên | Nhiệm vụ chính |
|---|---|
| Quang Lực | Nhóm trưởng, chốt đề tài, dựng sitemap/layout, tích hợp source, tổng hợp báo cáo |
| Cường | Trang chủ, CSS dùng chung, banner/logo, responsive giao diện chính |
| T. Dũng | Trang đăng ký, đăng nhập, validate form, LocalStorage người dùng |
| H. Thành | Dữ liệu PC thật, danh sách sản phẩm, tìm kiếm và lọc |
| H. Tài, T. Anh | Chi tiết sản phẩm, giỏ hàng, thanh toán, xác nhận đơn hàng |
| H. Anh | Trang giới thiệu, tin tức, chi tiết tin tức, nội dung website, kiểm thử |

## 3. Chức năng chính

- Trang chủ
- Sitemap
- Danh sách sản phẩm PC
- Chi tiết sản phẩm
- Tìm kiếm và lọc sản phẩm
- Tin tức và chi tiết tin tức
- Đăng ký, đăng nhập bằng LocalStorage
- Giỏ hàng
- Thanh toán mô phỏng
- Giới thiệu / liên hệ
- Responsive trên desktop, tablet, mobile

## 4. Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript ES2025
- Bootstrap 5 local
- jQuery 4 local
- LocalStorage
